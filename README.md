# EmotionRecognitionAI
# 🎤 Emotion Recognition AI

## 🚀 Project Overview

Emotion Recognition AI is an advanced Deep Learning web application that detects human emotions from speech audio using AI-powered speech analysis.

The system analyzes uploaded `.wav` audio files and predicts emotions such as:

- 😀 Happy
- 😢 Sad
- 😡 Angry
- 😨 Fear
- 😐 Neutral
- 🤢 Disgust
- 😲 Surprise

This project combines:
- Speech Signal Processing
- Deep Learning
- Audio Feature Engineering
- Interactive AI Dashboard

---

# 🧠 Deep Learning Architecture

The project uses an advanced hybrid architecture:

✅ CNN (Convolutional Neural Network)  
✅ LSTM (Long Short-Term Memory)  
✅ Batch Normalization  
✅ Dropout Regularization  

This architecture helps in:
- extracting audio patterns
- learning temporal speech dependencies
- improving prediction accuracy

# 📊 Audio Features Extracted

The AI model extracts advanced speech features including:

- MFCC (Mel Frequency Cepstral Coefficients)
- Chroma Features
- Mel Spectrogram Features

These features significantly improve emotion recognition accuracy.
